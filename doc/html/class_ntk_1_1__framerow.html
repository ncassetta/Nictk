Detailed Description

Internal class.

It represents a row of a NtkRowFrame container, and you can pack widgets horizontally inside it.

The constructor.

It sets some internal variables and calculates the row dimensions. When a _framerow is constructed it becomes the active row in the parent: widgets created are added to this row (see Ntk.NtkRowFrame).

Parameters
parentthe Ntk.NtkRowFrame to which the row belongs
hthe height of the row (its width coincides with that of the parent; see Placing the widgets for the various options you have
