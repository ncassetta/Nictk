Dialog boxes

In tkinter there are some functions to open the standard dialog boxes of the System, grouped in some submodules. Here is a list of them with their name in Ntk. For a reference with all their parameters see here.

For an example of their use see the file Dialogs.py.

Function Notes Returns
Nictk.mb.askokcancel Opens a dialog box with tho buttons "Ok", "Cancel". True for "Ok" and False for "Cancel".
Nictk.mb.askquestion Opens a dialog box with two buttons "Yes", "No". "yes" or "no".
Nictk.mb.askretrycancel Opens a dialog box with two buttons "Retry", "Cancel". True for "Retry" and False for "Cancel".
Nictk.mb.askyesno Same of askquestion. True for "Yes" and False for "No".
Nictk.mb.showerror Opens a dialog box with an error icon and only the button "Ok". Always "ok".
Nictk.mb.showinfo Opens a dialog box with an info icon and only the button "Ok". Always "ok".
Nictk.mb.showwarning Opens a dialog box with a warning icon and only the button "Ok". Always "ok".
Nictk.fd.askopenfilename Opens the file dialog "Open file" where you can choose a file. The complete path of the chosen file, or an empty string if you press "Cancel".
Nictk.fd.asksaveasfilename Opens the file dialog "Save as file" where you can choose a file. The complete path of the chosen file, or an empty string if you press "Cancel".
Nictk.cc.askcolor Opens a color chooser file dialog. A duple, with in the first element three integers R, G, B, in the second a tkinter Color object, or (None, None) if you press "Cancel"
