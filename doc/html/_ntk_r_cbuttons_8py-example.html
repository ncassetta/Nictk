NtkRCbuttons.py

An example of the use of classes Ntk.NtkCheckbutton and Ntk.NtkRadiobutton widgets. They have both a Variable object associated (which automatically updates a label) and a callback with command option. We used a Ntk.NtkRowFrame and the strings "pack", "fill", etc. (alternative to constants PACK, FILL ...) for positioning the widgets.
