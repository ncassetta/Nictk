Available widgets

This is a list of the available widgets in the library; all of them are subclasses of tkinter or ttk widgets. I however renamed some of them to get a better clarity.

Windows

In your app you must create a unique main window (i.e. a Main object), then you can create and destroy any number of Window objects (the Toplevel object in tkinter).

Object
class Nictk.Main(Nictk.BaseWindow, tk.Tk) The main window of the app (derived from the tkinter Tk class)
class Nictk.Window(Nictk.BaseWindow, tk.Toplevel) A common window (derived from the tkinter Toplevel class). It can be a normal, modal or persistent window

Frames

Frames are another type of container. They are initialized with no border and the same color of their parent, so they are invisible and you can use them for layout purposes (grouping children widgets horizontally or vertically). However, as they are derived from the tkinter LabelFrame widget, you can set a border and a label for them (this reduces the available space in the frame). In their constructors the pad parameter is present but is ignored (it is always fixed to 0 in all directions).

Object
class Nictk.HorFrame(Nictk.Widget, Nictk.Container, tk.LabelFrame) A frame in which you can stack children widgets horizontally
class Nictk.VerFrame(Nictk.Widget, Nictk.Container, tk.LabelFrame) A frame in which you can stack children widgets vertically
class Nictk.RowFrame(Nictk.Widget, Nictk.Container, tk.LabelFrame) A frame which you can divide in rows vertically. Each row behaves like a HorFrame, allowing to stack children widgets horizontally. This allows to place widgets as in a matrix.

Widgets

These are the other widgets defined in Nictk.

Object
class Nictk.Button(Nictk.Widget, tk.Button) Button widget
class Nictk.Canvas(Nictk.Widget, tk.Canvas) Canvas widget to display graphical elements like lines or text
class Nictk.Checkbutton(Nictk.Widget, tk.Checkbutton) Checkbutton widget which is either in on or off state
class Nictk.Combobox(Nictk.Widget, tk.OptionMenu) Combobox which allows the user to select a value from a menu. It is the equivalent (renamed) of the OptionMenu class in tkinter
class Nictk.Entry(Nictk.Widget, tk.Entry) Entry widget which allows editing and displaying simple text
class Nictk.Label(Nictk.Widget, tk.Label) Label widget which can display text and bitmaps
class Nictk.Listbox(Nictk.Widget, tk.Listbox) Listbox widget which can display a list of strings, allowing simple and multiple selection
class Nictk.Menu(tk.Menu) Menu widget which allows displaying menu bars, pull-down menus and pop-up menus
class Nictk.Radiobutton(Nictk.Widget, tk.Radiobutton) Radiobutton widget which can be grouped, showing only one of several buttons in on state
class Nictk.Scale(Nictk.Widget, tk.Scale) Scale widget which can display a cursor sliding on a numerical scale
class Nictk.Scrollbar(tk.Scrollbar) Scrollbar widget
class Nictk.Spinbox(Nictk.Widget, tk.Spinbox) Spinbox widget which allows selecting one of various options by mean of arrow keys or inserting a new one
class Nictk.Text(Nictk.Widget, tk.Text) Text widget which can edit and display formatted text
class Nictk.Notebook(Nictk.Widget, ttk.Notebook) Notebook widget which manages a collection of windows and displays a single one at a time.
