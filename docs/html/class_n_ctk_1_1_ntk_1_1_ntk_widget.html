◆ hide()

def hide (   self)

Hides the widget.

The widget will not be displayed, but its data remain managed; use show() to newly display the widget. if the widget was already hidden it does nothing.

◆ show()

def show (   self)

Shows a previously hidden widget.

If the widget was already shown it does nothing.
