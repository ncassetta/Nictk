Available widgets

This is a list of the available widgets in the library; all of them are subclasses of tkinter or ttk widgets. I however renamed some of them to get a better clarity.

Windows

In your app you must create a unique main window (i.e. a NtkMain object), then you can create and destroy any number of NtkWindow objects (the Toplevel object in tkinter).

Object
class Ntk.NtkMain(Ntk.BaseWindow, tk.Tk) The main window of the app (derived from the tkinter Tk class)
class Ntk.NtkWindow(Ntk.BaseWindow, tk.Toplevel) A common window (derived from the tkinter Toplevel class). It can be a normal, modal or persistent window

Frames

Frames are another type of container. They are initialized with no border and the same color of their parent, so they are invisible and you can use them for layout purposes (grouping children widgets horizontally or vertically). However, as they are derived from the tkinter LabelFrame widget, you can set a border and a label for them (this reduces the available space in the frame). In their constructors the pad parameter is present but is ignored (it is always fixed to 0 in all directions).

Object
class Ntk.NtkHorFrame(Ntk.NtkWidget, Ntk.NtkContainer, tk.LabelFrame) A frame in which you can stack children widgets horizontally
class Ntk.NtkVerFrame(Ntk.NtkWidget, Ntk.NtkContainer, tk.LabelFrame) A frame in which you can stack children widgets vertically
class Ntk.NtkRowFrame(Ntk.NtkWidget, Ntk.NtkContainer, tk.LabelFrame) A frame which you can divide in rows vertically. Each row behaves like a NtkHorFrame, allowing to stack children widgets horizontally. This allows to place widgets as in a matrix.

Widgets

These are the other widgets defined in Ntk.

Object
class Ntk.NtkButton(Ntk.NtkWidget, tk.Button) Button widget
class Ntk.NtkCanvas(Ntk.NtkWidget, tk.Canvas) Canvas widget to display graphical elements like lines or text
class Ntk.NtkCheckbutton(Ntk.NtkWidget, tk.Checkbutton) Checkbutton widget which is either in on or off state
class Ntk.NtkCombobox(NtkWidget, tk.OptionMenu) Combobox which allows the user to select a value from a menu. It is the equivalent (renamed) of the OptionMenu class in tkinter
class Ntk.NtkEntry(Ntk.NtkWidget, tk.Entry) Entry widget which allows editing and displaying simple text
class Ntk.NtkLabel(Ntk.NtkWidget, tk.Label) Label widget which can display text and bitmaps
class Ntk.NtkListbox(Ntk.NtkWidget, tk.Listbox) Listbox widget which can display a list of strings, allowing simple and multiple selection
class Ntk.NtkMenu(tk.Menu) Menu widget which allows displaying menu bars, pull-down menus and pop-up menus
class Ntk.NtkRadiobutton(Ntk.NtkWidget, tk.Radiobutton) Radiobutton widget which can be grouped, showing only one of several buttons in on state
class Ntk.NtkScale(Ntk.NtkWidget, tk.Scale) Scale widget which can display a cursor sliding on a numerical scale
class Ntk.NtkScrollbar(tk.Scrollbar) Scrollbar widget
class Ntk.NtkSpinbox(NtkWidget, tk.Spinbox) Spinbox widget which allows selecting one of various options by mean of arrow keys or inserting a new one
class Ntk.NtkText(NtkWidget, tk.Text) Text widget which can edit and display formatted text
class Ntk.NtkNotebook(NtkWidget, ttk.Notebook) Notebook widget which manages a collection of windows and displays a single one at a time.
