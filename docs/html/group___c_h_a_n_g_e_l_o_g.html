Version 2.0.0 (May 14, 2022)

Renamed Ntk into Nictk and published it as a Python package (with an __init__.py main file). Dropped the Ntk prefix from widget names, so this version is incompatible with 1.0.1. Also the example files were renamed without the Ntk prefix.

    Version 2.1.0 (Jun 25, 2023)

Some improvements and bugfixes

Bugfix

    Version 2.1.1 (Feb 11, 2024)

Some improvements and bugfixes

Bugfix

  • (__init__.py L484) Corrected: in Widget.calc_dimensions the Canvas returns borderwidth as a string, not a number
  • (__init__.py L1360) Corrected: calling Combobox.get_items() raises an error
  • (__init__.py L1428-1429) Corrected: now in Combobox.delete() if the selected item is cancelled the selection is reset
  • (__init__.py L2198) Corrected: calling the Scrollbar constructor raises an error

New features

  • (__init__.py L786-788) Added the method Window.visible()
  • (__init__.py L1431-1434) Added the method Combobox().clear()
  • (__init__.py L1866-1891 L1946-1950) Added the methods Menu.activate(), Menu.deactivate(), Menu.entry_activate(), Menu.entry_deactivate(), Menu.clear()
  • (__init__.py L27) The import of IntVar, StringVar, DoubleVar, BooleanVar, mainloop, EventType was moved from constants.py to __init__.py
  • (constants.py L29-30) Imported all dialog boxes related constants

  • (__init__.py L1399 L1413) Now methods Combobox.add() and Combobox.insert() accept a list/tuple of items
  • (__init__.py L1469) Now method Combobox.index() raises an error if the argument is not in the menu
  • (__init__.py L1145-1146) Added the method Canvas.resize_children for future development (currently it does nothing)

Examples
