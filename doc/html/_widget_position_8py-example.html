WidgetPosition.py

An example of the various modes to position a widget inside a parent container (remember that NtkMain and NtkWindow behave as vertical frames, i.e. pack widgets vertically). You can try to maximize, iconize and resize the window (dragging its borders with the mouse) to see the various behaviors of widgets when the parent is resized.
