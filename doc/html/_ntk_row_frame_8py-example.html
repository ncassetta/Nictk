NtkRowFrame.py

An example of the use of the NtkRowFrame container. You can add to it rows (with the add_row() method) which behave as horizontal frames (i.e. you can pack widget horizontally). You can set rows height as absolute, relative (percentual) or "absolute from bottom" values.
